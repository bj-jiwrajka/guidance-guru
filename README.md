## Getting Started

1. Clone the repository:
   ```bash
   git clone https://github.com/bj-jiwrajka/guidance-guru.git
   ```
2. Navigate to the project folder:
    ```bash
      cd guidance-guru
    ```
    
3. Install dependencies:
    ```
    bash
    npm install
    ```
    
4. Start the app:
   ``` 
    bash
    npm run dev
   ```
